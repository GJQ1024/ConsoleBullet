# 弹幕游戏demo

## 项目简介
这是一个基于C#的控制台弹幕游戏，玩家将与Boss进行激烈的对抗，通过躲避和射击来生存并击败Boss。

## 功能特点
- **动态弹幕**：Boss会发射弹幕攻击玩家，弹幕模式会随时间变化。
- **玩家控制**：玩家可以控制角色左右移动并发射子弹。
- **游戏状态显示**：实时显示玩家和Boss的HP以及游戏得分。
- **难度递增**：Boss的行为模式会随时间变化，增加游戏难度。
